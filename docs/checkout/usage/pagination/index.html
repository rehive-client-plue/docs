Pagination

On listing pages, the API provides a way to paginate results. The default pagination method offered by the API is offset pagination, which allows navigation to an arbitrary point in a list of results as well as via next and prevattributes. The JSON format returned on paginated listing pages looks like this:

 {
    "status": "success",
    "data": {
        "count": 1,
        "next": null,
        "previous": null,
        "results": []
    }
}

The default page size is 15 but can be changed by adding a page_size query parameter to the request URL.
